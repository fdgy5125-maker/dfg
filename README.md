# Mikrotik Manager - مدير الميكروتك

تطبيق سطح مكتب متقدم لإدارة أجهزة الميكروتك والاشتراكات والفواتير.

## المميزات

✅ **إدارة الأجهزة**
- إضافة وتعديل وحذف أجهزة الميكروتك
- تتبع حالة الأجهزة (متصل/غير متصل)
- عرض معلومات الجهاز (IP، الموديل، الموقع)
- سجل النشاط والعمليات

✅ **إدارة الاشتراكات**
- خطط مختلفة للاشتراكات
- تتبع عدد الأجهزة المسموح بها
- إنشاء وتحديث الفواتير
- تتبع حالة الدفع

✅ **الأمان والخصوصية**
- مصادقة آمنة مع Supabase
- تشفير البيانات
- السياسات الأمنية (RLS)
- كل مستخدم يرى بيانته فقط

✅ **واجهة سهلة الاستخدام**
- تصميم عصري وأنيق
- دعم اللغة العربية
- واجهة متجاوبة
- سرعة عالية

## المتطلبات

- Node.js 16+
- npm أو yarn

## التثبيت

```bash
npm install
```

## التطوير

```bash
npm run dev
```

للقيام بتشغيل تطبيق Electron أثناء التطوير:

```bash
npm run electron-dev
```

## البناء

```bash
npm run build
```

```bash

```

- **Lucide React** - أيقونات جميلة

## قاعدة البيانات

### الجداول الرئيسية

1. **devices** - أجهزة الميكروتك
   - المعلومات الأساسية (الاسم، IP، الموديل)
   - حالة الجهاز (متصل/غير متصل)
   - مقاييس الأداء

2. **users** - ملفات المستخدمين
   - البيانات الشخصية
   - معلومات الشركة
   - حالة الاشتراك

3. **subscriptions** - الاشتراكات
   - نوع الخطة
   - عدد الأجهزة المسموح بها
   - تاريخ الانتهاء
   - المميزات الإضافية

4. **invoices** - الفواتير
   - المبلغ والعملة
   - حالة الدفع
   - تاريخ الإصدار والدفع

5. **device_logs** - سجل العمليات
   - تتبع جميع العمليات
   - التاريخ والتفاصيل

6. **device_status** - حالة الجهاز الفورية
   - استخدام CPU والذاكرة
   - سرعة الاتصال
   - الاتصالات النشطة

## متغيرات البيئة

قم بإنشاء ملف `.env` بالبيانات التالية:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
