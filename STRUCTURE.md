# بنية مشروع مدير الميكروتك

## المشروع
تطبيق سطح مكتب متقدم لإدارة أجهزة الميكروتك مع مميزات الاشتراكات والفواتير.

## البنية الأساسية

```
.
├── src/                           # كود المصدر
│   ├── components/                # مكونات React
│   │   ├── Login.jsx             # صفحة تسجيل الدخول/التسجيل
│   │   ├── Dashboard.jsx         # لوحة التحكم الرئيسية
│   │   ├── Statistics.jsx        # عرض الإحصائيات
│   │   └── Invoices.jsx          # إدارة الفواتير
│   ├── services/                 # خدمات البيانات
│   │   ├── supabase.js          # تهيئة Supabase
│   │   ├── auth.js              # خدمات المصادقة
│   │   ├── devices.js           # عمليات الأجهزة
│   │   └── subscriptions.js      # عمليات الاشتراكات
│   ├── styles/                   # أنماط CSS
│   │   └── App.css              # الأنماط الرئيسية
│   ├── App.jsx                   # المكون الرئيسي
│   └── main.jsx                  # نقطة الدخول
├── public/                        # الملفات الثابتة
│   └── electron.js              # تكوين Electron
├── dist/                          # ملفات البناء (generated)
├── node_modules/                 # المكتبات
├── package.json                  # معلومات المشروع
├── vite.config.js               # تكوين Vite
├── index.html                    # صفحة HTML الرئيسية
├── README.md                     # وثائق المشروع
└── .gitignore                    # ملفات git المتجاهلة
```

## المكونات الرئيسية

### Login.jsx
- تسجيل الدخول للمستخدمين الموجودين
- إنشاء حسابات جديدة
- معالجة الأخطاء

### Dashboard.jsx
- العنصر الرئيسي بعد تسجيل الدخول
- تابات للتنقل بين الأقسام
- إدارة الأجهزة والإحصائيات والفواتير

### Statistics.jsx
- عرض الإحصائيات العامة
- عدد الأجهزة والحالة
- معلومات الاشتراك

### Invoices.jsx
- قائمة الفواتير
- تصفية الفواتير (مدفوعة/معلقة)
- تحديث حالة الفواتير

## خدمات البيانات

### supabase.js
- تهيئة عميل Supabase
- إدارة الاتصال بقاعدة البيانات

### auth.js
- تسجيل الدخول والخروج
- إنشاء حسابات جديدة
- إدارة ملفات المستخدم

### devices.js
- إضافة وتحديث وحذف الأجهزة
- الحصول على قائمة الأجهزة
- إدارة سجلات الأجهزة

### subscriptions.js
- إدارة الاشتراكات
- إدارة الفواتير
- تحديث حالة الدفع

## قاعدة البيانات

### الجداول

1. **devices**
   - معلومات أجهزة الميكروتك
   - الحالة والأداء

2. **users**
   - بيانات المستخدمين
   - معلومات الحساب

3. **subscriptions**
   - خطط الاشتراك
   - حدود الأجهزة

4. **invoices**
   - الفواتير والدفعات
   - سجل المدفوعات

5. **device_logs**
   - سجل العمليات على الأجهزة

6. **device_status**
   - معلومات الأداء الفورية

## الأمان

- المصادقة آمنة مع Supabase Auth
- تشفير كلمات المرور
- Row Level Security (RLS) على جميع الجداول
- كل مستخدم يرى بيانته فقط
